Sobre o Projeto

Esse projeto foi utilizados varios codigos de Html, para fazer todo o esqueleto do site, então entrei com o JavaScript dando funcionalidade as coisas, além também do CSS3 para fornecer estilo e deixar a pagina mais aconchegante, desta forma mantendo a beleza e transmitindo todo o amor e mensagem que eu queria

Foi um dos meus primeiros projetos, dei inicio a ele no começo da faculdade, usei o mesmo para me aprofundar no font-end e agora estou migrando para o backend com destino a ser full stack, aprendi sobre React e JavaScript nesse codigo, para saber mais sobre ele é só olhar aqui embaixo.
