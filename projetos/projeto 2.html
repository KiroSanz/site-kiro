Sobre o Projeto

Esse projeto foi utilizados varios codigos de Html, para fazer todo o esqueleto do site, então entrei com o JavaScript dando funcionalidade as coisas, além também do CSS3 para fornecer estilo e deixar a pagina mais aconchegante, desta forma mantendo a beleza e transmitindo a seriedade e profissionalismo

Foi uma das minhas primeiras vendas, iniciei o desenvolvimento dele no final de 2024 e o entreguei com todas as funcionalidades semanas após, aprendi sobre Banco de Dados, formulários e outras coisas nesse codigo, para saber mais sobre ele é só olhar aqui embaixo.
